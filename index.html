Providing a digital comminication channel for rental tenants.

repo: github.com/jamespankhurst/letcomm

In the pre-internet ninties, I was involved in developing insurance claim software, with particular reference to property maintenance.

I developed the first web version of this software in classic ASP, claimtrack.co.uk... re-coded for .NET to produce; trackpro.co.uk,the current version.

Even though the system was now internet based, the benefits remained with the landlords/property management companies (PMCs).

They had the ability to allow tenant access, but rarely, if ever, granted it.

The difference with letcomm is that the TENANT controls access.
